#Welcome to Seat Yo-Self

This is a project modelled off the popular site Open Table. 
Users can add new restaurants, and set and view reservations. 


###Functionality Highlights

* Utilizes nested resources 

* Single user model for both creating reservations and restaurants

###Specs

* Made with Ruby on Rails(infrastructure)
